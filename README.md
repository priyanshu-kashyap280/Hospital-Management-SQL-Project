# 🏥 Hospital Management SQL Project

A complete **Hospital Management System** built using **MySQL** to demonstrate real-world SQL skills and healthcare data analysis.

This project covers database design, sample datasets, SQL queries, and business insights that help analyze hospital operations such as patient management, doctor performance, appointments, treatments, and billing.

---

## 📌 Project Objective

The main objective of this project is to analyze hospital data using SQL and generate meaningful insights that support better decision-making in healthcare management.
